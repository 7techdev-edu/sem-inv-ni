## 📌 Cómo usar este repositorio

1. Navega a la sección que necesitas usando los enlaces de arriba.
2. **Para prompts:** copia el texto y pégalo en tu herramienta de IA (Claude, ChatGPT, etc.), reemplazando los parámetros entre corchetes `[COMO ESTE]`.
3. **Para scripts:** sigue las instrucciones del entorno de ejecución indicadas en cada archivo antes de correr el código.
